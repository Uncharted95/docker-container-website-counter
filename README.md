**Instructions to compile and run the application:**

By accessing the localhost (Port: 5000:5000) the user can access the website which will update the counter data per visit.


**Description of technical choices (base images, networking, optimering):** 

This application platform uses Python (3:13:11). This docker container uses a flask template file, which dynamically fills in the HTML file with data before uploading to the browser.
